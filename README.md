# 💼 Job Finder App

A modern job search application built with React and Tailwind CSS.

## 🚀 Features
- 🔍 Search real jobs (Remotive API)
- 📄 Job detail view
- ❤️ Favorite jobs
- 🌙 Dark mode
- 💾 Persistent data (localStorage)

## 🛠 Tech Stack
- React
- Tailwind CSS
- Vite
